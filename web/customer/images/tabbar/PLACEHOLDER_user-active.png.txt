# 这是 user-active.png 图标文件的占位符

请将准备好的图标文件命名为 user-active.png，替换此文件。

## 图标要求

- **文件名**: user-active.png（必须完全匹配）
- **尺寸**: 81px × 81px
- **格式**: PNG（支持透明背景）
- **颜色**: 咖啡色 (#6B4423 或 #8B5A2B)
- **功能**: 我的按钮的选中状态图标
- **建议图案**: 用户图标 或 人像图标（与 user.png 相同图案，仅颜色不同）

## 如何获取图标

1. 访问 https://www.iconfont.cn/
2. 搜索 "user" 或 "用户"
3. 选择与 user.png 相同的图标
4. 下载 PNG 格式，尺寸 81px
5. 颜色设置为咖啡色 (#6B4423)
6. 重命名为 user-active.png
7. 替换此文件
